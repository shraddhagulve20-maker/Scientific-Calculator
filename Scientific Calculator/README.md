Scientific Calculator
Overview
basic calculator app i made using python. has normal calculator stuff and also scientific functions like sin cos tan etc.
Features

basic math (add subtract multiply divide)
scientific stuff (sin, cos, tan, log, square root)
constants like pi and e
can switch between simple mode and scientific mode
clear buttons to reset

Tech Used

Python 3
Tkinter for GUI
math module

How to Run

install python if you dont have it
download the calculator.py file
run it:

   python calculator.py
thats it, no extra packages needed
Testing
try these to see if it works:

5 + 3 = should give 8
90 then sin = should give 1
pi button = shows 3.14...
2 x^y 3 = should give 8
